# 技术架构

## 系统组件

### 1. 代理管理系统
- **文件**: `modules/proxy_manager.py`
- **功能**: 轮换IP地址以避免检测
- **特性**:
  - 多代理提供商支持
  - 健康监控
  - 故障转移机制
  - 统计跟踪

### 2. 验证码解决集成
- **文件**: `modules/captcha_solver.py`
- **功能**: 与验证码解决服务集成
- **特性**:
  - 图片验证码解决
  - reCAPTCHA v2/v3 支持
  - 轮询机制
  - 超时处理

### 3. 邮箱处理系统
- **文件**: `modules/email_handler.py`
- **功能**: 管理临时邮箱账户进行验证
- **特性**:
  - Guerrilla Mail API 集成
  - 邮箱监控
  - 验证链接提取
  - 超时机制

### 4. 浏览器自动化
- **文件**: `modules/twitter_bot.py`
- **功能**: Twitter交互的主要自动化引擎
- **特性**:
  - 隐身浏览器初始化
  - 账户创建工作流
  - 登录处理
  - 帖子创建
  - 指纹欺骗

### 5. 工作流编排
- **文件**: `modules/workflow_manager.py`
- **功能**: 协调所有组件
- **特性**:
  - 端到端过程管理
  - 错误处理和恢复
  - 重试机制
