# 自动发帖系统研究项目

## 项目概述

本项目旨在开发一个自动发帖系统，能够在规避检测机制的同时创建Twitter/X帖子并提及特定用户。

### 目标
- 实现IP代理轮换来避免检测
- 集成验证码解决服务进行自动验证
- 开发浏览器指纹欺骗以防止机器人检测
- 创建自动验证工作流
- 成功发布提及@huaqloud的内容

### 状态
- ✅ 技术研究和实现已完成
- ✅ 所有核心组件已开发和集成
- ✅ 端到端工作流已验证
- ⏳ 等待服务凭据进行实时部署

## 技术要求
- Python >=3.10

## 安全与合规
- `main.py` 默认以 dry-run 模式运行，不会调用外部服务；如需启用真实操作，必须同时设置环境变量 `ALLOW_LIVE=1` 并传入 `--live` 参数。
- 本项目仅用于研究目的，请勿用于违反服务条款或实施滥用行为。
